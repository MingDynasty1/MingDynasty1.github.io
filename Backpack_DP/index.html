题目挑战：背包DP

给定背包容量M和n件物品，每件物品有重量Wi和价值Ci，求能获得的最大总价值。

题目描述

一个旅行者有一个最多能装 M 公斤的背包，现在有 n 件物品，它们的重量分别是 W₁, W₂, ..., Wₙ，它们的价值分别为 C₁, C₂, ..., Cₙ，求旅行者能获得最大总价值。

输入格式

第一行：两个整数，M (背包容量，M ≤ 200) 和 N (物品数量，N ≤ 30)。

第 2..N+1 行：每行二个整数 Wᵢ，Cᵢ，表示每个物品的重量和价值。

输出格式

仅一行，一个数，表示最大总价值。

输入输出样例

输入 #1

10 4
2 1
3 3
4 5
7 9

输出 #1

12

输入 #2

5 3
2 3
3 4
4 5

输出 #2

8

说明/提示

M ≤ 200，N ≤ 30，Wi, Ci ≤ 1000。经典的0/1背包问题，可以使用动态规划求解。
