洛谷 P12572 题解

题目描述

给定一个字符串，需要过滤掉其中的特定字符：'1', 'i', 'l', 'I'，然后输出处理后的字符串。

查看原题

解题思路

本题考查分支结构和字符串处理。

基本思路是遍历字符串的每个字符，如果遇到需要过滤的字符则跳过，否则输出该字符。

步骤 1

读入字符串数量

步骤 2

循环处理每个字符串

步骤 3

遍历字符串中的每个字符

步骤 4

如果字符是 '1', 'i', 'l', 'I' 则跳过，否则输出

代码实现

C++ 代码 
/*
    Auther:MingDynasty
    Problem:https://www.luogu.com.cn/problem/P12527
*/
#include<bits/stdc++.h>
using namespace std;
int a,b,c;
int main(){
    cin.tie(0)->sync_with_stdio(0);
    cin>>a;
    while(a--){
        string k;
        cin>>k;
        for(int i=0;i<k.size();i++){
            if(k[i]=='1'||k[i]=='i'||k[i]=='l'||k[i]=='I') continue;
            cout<<k[i];
        }
        cout<<'\n';
    }
    return 0;
}

代码说明：

  • 使用 cin.tie(0)->sync_with_stdio(0) 加速输入输出
  • 循环读取每个字符串
  • 遍历字符串中的每个字符，过滤特定字符
  • 输出处理后的字符串

复杂度分析

时间复杂度：O(n)，其中 n 是所有字符串的总长度

空间复杂度：O(1)，除了存储输入外没有使用额外空间
