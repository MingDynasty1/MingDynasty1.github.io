## 🌏 网站简介

**大明史料文库**（[https://mingdynasty1.github.io](https://mingdynasty1.github.io)）乃当代学人精心构建之数字史籍平台，致力于汇集明朝（1368-1644）相关史料、研究成果与文化遗产，以现代科技手段传承华夏文明之瑰宝。

本网站秉持"辨章学术，考镜源流"之旨，力求为研究者、爱好者提供一站式明朝历史文化资源，重现大明风华，弘扬中华传统。

## 📚 主要内容

### 1. 史籍典藏
- **正史汇要**：《明史》《明实录》《明通鉴》等官修史料数字化呈现
- **私家著述**：收录明代文人笔记、奏疏、文集等私家史料
- **域外史料**：整理传教士、外国使者关于明朝之记载

### 2. 文化艺术
- **书画艺术**：明代书画作品赏析与研究资料
- **陶瓷工艺**：景德镇窑业发展与青花瓷艺术
- **文学成就**：四大奇书、戏曲、诗文等文学瑰宝
- **服饰制度**：明代冠服制度与社会风尚

### 3. 政治经济
- **典章制度**：明代政治制度、科举考试、军事体系研究
- **经济发展**：江南资本主义萌芽、海外贸易与白银流入
- **城市变迁**：明代城市发展与市井生活

### 4. 科技军事
- **科技成就**：《天工开物》《本草纲目》等科技巨著研究
- **军事技术**：火器发展、神机营与边防体系
- **郑和下西洋**：航海技术与中外交流

## 🎯 网站特色

- **史料丰富**：汇集多种来源之明代史料，经严格校勘整理
- **分类系统**：采用现代分类学方法，便于检索与研究
- **学术性强**：依托专业历史学者团队，确保内容准确性与深度
- **用户友好**：界面设计简洁直观，支持多种检索方式
- **多媒体呈现**：结合图片、音频、视频等多种形式展示历史场景

## 🚀 使用指南

### 浏览方式
1. **分类导航**：通过顶部导航栏按类别浏览内容
2. **关键词搜索**：使用搜索框输入关键词查找相关资料
3. **时间轴浏览**：按明代历史时间线探索重要事件与人物

### 资源下载
- 部分史料支持PDF格式下载
- 图片资源可高清查看与保存
- 学术论文提供引用格式

### 互动功能
- 注册用户可收藏感兴趣的内容
- 参与学术讨论与问答
- 提交新史料与研究成果
